Часто используемые ссылки

PlaningPoker

planitpoker.com

Покер для оценки задач на планировании спринта. Для работы зарегистрированным должен быть только один человек, который модерирует процесс. Остальным можно просто ввести имя при подключении кпроцессу планирования.

Trello

trello.com

Простой инструмент по работе с карточками на одной доске в реальном времени. Для совместной работы каждый должен быть зарегистрирован. Для регистрации можно использовать гугл аккаунт.

TrelloExport

TrelloExport for Chrome

Планин для браузера хром, позволяющий выгружать в csv(excel) формате данные из трелло.
