PlaningPoker

planitpoker.com

Покер для оценки задач на планировании спринта. Для работы зарегистрированным должен быть только один человек, который модерирует процесс. Остальным можно просто ввести имя при подключении кпроцессу планирования.

Trello

trello.comTrelloExport for ChromeCustom Fields for Trello

Простой инструмент по работе с карточками на одной доске в реальном времени. Для совместной работы каждый должен быть зарегистрирован. Для регистрации можно использовать гугл аккаунт.

TrelloExport - Планин для браузера хром, позволяющий выгружать в csv(excel) формате данные из трелло.

Custom Fields for Trello - Планин для досок трелло, позволяющий добавлять преднастроенные поля на карточках. Добавляется к каждой доске отдельно.

Retromat

retromat.org

Планируете вашу следующию аджайл ретроспективу? Начните со случайного плана, адаптируйте его к вашей команде и ситуации, распечатайте его или поделитесь URL вашего плана с другими. Или просто полистайте в поиске новых идей!

Canvas

Lean CanvasЧто такое Lean Canvas?

Lean Canvas (можно перевести как «бережливый шаблон бизнес модели») — это шаблон модели для бизнеса, помещает все важнейшие элементы бизнеса на одну страницу, помогая вам изложить ключевую информацию, которая вам нужна, без каких-либо лишних деталей.
